== Description ==

WP Link Hover Effect plugin By MasterBlogster adds cool hover effect for links in the wordpress post page. Hover the link to see the effect. 

Check out the demo at <a href="http://masterblogster.com">Master Blogster</a>

Other famous sites using this plugin are-

<ul>
<li> http://techgyo.com </li>
<li> http://gadgetnote.com </li>
</ul>


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
